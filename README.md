# TG Grades Bot

Бот для учёта оценок за тесты, статистики и лидерборда.

## Возможности
- Учёт оценок и статистика
- Лидерборд
- Система заявок/доступов (если включено в текущей версии)
- Админ-обработка запросов (принять/отклонить)

## Быстрый запуск

### 1) Установка зависимостей
Рекомендуется Python 3.10+

```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
# source .venv/bin/activate

pip install -r requirements.txt
