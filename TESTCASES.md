\# Тест-кейсы



1\. Регистрация

\- Шаги: /start → "Иван Иванов"

\- Ожидаемо: пользователь зарегистрирован, показано меню



2\. Добавление оценки

\- Шаги: "Добавить оценку" → "Русский" → "5"

\- Ожидаемо: оценка сохранена, сообщение об успехе



3\. Добавление дробной оценки

\- Шаги: "Добавить оценку" → "Русский" → ввод "4,35"

\- Ожидаемо: сохранено 4.35



4\. Личный кабинет без оценок

\- Шаги: зарегистрировать нового пользователя → "Личный кабинет"

\- Ожидаемо: средняя “—”, подсказка добавить оценки



5\. Личный кабинет с оценками

\- Шаги: добавить 3 оценки → "Личный кабинет"

\- Ожидаемо: общая средняя и средние по предметам корректны



6\. Лидерборд

\- Шаги: несколько пользователей добавили оценки → "Лидерборд"

\- Ожидаемо: сортировка по средней, топ-10



7\. Удаление одной оценки

\- Шаги: "Удалить одну" → выбрать ID → удалить

\- Ожидаемо: оценка исчезает из статистики



8\. Удаление всех оценок

\- Шаги: "Удалить все" → написать "ДА"

\- Ожидаемо: все оценки удалены



9\. Админка доступ

\- Шаги: пользователь не админ → нажать "Админка"

\- Ожидаемо: отказ в доступе



10\. Админ удаляет пользователя

\- Шаги: админ → удалить пользователя по TG ID

\- Ожидаемо: пользователь и оценки удалены
