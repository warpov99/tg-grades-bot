\# Техническое задание (ТЗ) — TG Grades Bot



\## Цель

Создать Telegram-бота для учёта оценок за тесты, формирования статистики и лидерборда.



\## Функциональные требования

1\. Регистрация пользователя по Имя Фамилия.

2\. Добавление оценки:

&nbsp;  - выбор предмета из списка

&nbsp;  - возможность добавить новый предмет

&nbsp;  - ввод оценки (в т.ч. дробной)

3\. Личный кабинет:

&nbsp;  - общая средняя

&nbsp;  - средняя по каждому предмету

&nbsp;  - количество оценок

4\. Лидерборд:

&nbsp;  - топ-10 по общей средней

5\. Удаление оценок пользователем:

&nbsp;  - удалить одну оценку по ID

&nbsp;  - удалить все оценки (с подтверждением)

6\. Админка (по TG ID):

&nbsp;  - список пользователей

&nbsp;  - удаление пользователя

&nbsp;  - добавление демо-пользователей

&nbsp;  - добавление оценки пользователю

&nbsp;  - удаление оценок пользователю



\## Нефункциональные требования

\- хранение данных в SQLite

\- логирование событий в консоль

\- безопасная работа с токеном (не публиковать)
